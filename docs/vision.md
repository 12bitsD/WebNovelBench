# WebNovelBench v3: 核心设计文档

> 本文档是项目的唯一设计权威。
> 所有实现必须遵循本文档定义的评估维度、评分标准和 Judge 行为规范。

---

## 一、项目定位

### 我们在做什么

构建一个评估体系，用于判断一部中文网文/长篇故事的**叙事工艺水平**。

### 我们不在做什么

- 不是"读者喜好度"预测（那受市场、类型偏好、个人口味影响）
- 不是"文学价值"评判（那需要历史语境和学术共识）
- 不是统计分析（不数词、不数句、不数段落长度）

### 核心问题

一部作品的叙事机器是否精密——
作者对信息释放、角色塑造、结构张力、主题表达的控制精度如何？

---

## 二、设计原则

### 原则 1：Agent-as-Judge，不是统计

评估方式：Agent 阅读文本 → 应用详细 Rubric → 给出带证据的评分。

不使用任何统计指标（词频、句长、段落数、n-gram 等）。
原因：统计指标测的是文本的表面特征，不是叙事效果。
一个反转是否"意料之外"、角色弧光是否"被故事驱动"——
这些需要理解上下文，不是数词能解决的。

### 原则 2：Rubric 决定一切

Agent-as-Judge 的可靠性 = Rubric 的质量。

Rubric 必须满足：
  a) 每个分数段有明确的、可观察的文本特征
  b) 有正面例子（这个分数应该是什么样的）
  c) 有反面例子（这个分数不应该是什么样的）
  d) 有边界案例的判断规则（当两个分数都可能时，怎么选）
  e) 要求引用具体文本证据（不能凭感觉打分）

### 原则 3：维度正交，不重叠

10 个评估维度，每个维度测量独立的叙事能力。
维度之间可能相关（好的作品往往多个维度都好），
但评估时必须独立打分，不能因为一个维度好就默认另一个也好。

### 原则 4：过程可追溯

每个评分必须包含：
  - 分数
  - 判断依据（具体观察到了什么）
  - 文本证据（引用原文）
  - 判断推理（从观察到分数的逻辑链）

---

## 三、评估维度定义

### 总览

| # | 维度 | 测量什么 | 核心问题 |
|---|------|---------|---------|
| D1 | 信息流管理 | 信息释放的节奏和精密程度 | 读者的好奇心是否被持续且精确地操控？ |
| D2 | 角色弧光 | 角色变化的深度和合理性 | 角色的变化是否被故事逻辑驱动，是否与主题共鸣？ |
| D3 | 世界观构建 | 世界的系统性和可信度 | 世界观是否自洽、有深度、通过行动而非旁白展示？ |
| D4 | 主题共振 | 主题的深度和表达方式 | 主题是否通过结构自然呈现，而非通过角色说教？ |
| D5 | 结构张力 | 节奏控制和冲突升级 | 张力是否有层次、有升级、有真正的代价？ |
| D6 | 颠覆与反转 | 对读者预期的操控精度 | 反转是否"意料之外，情理之中"？ |
| D7 | 展示艺术 | "展示而非讲述"的执行度 | 世界观/情感/冲突是否通过行动展示？ |
| D8 | 冲突设计 | 冲突的多元性和策略性 | 冲突是否有层次、有策略、不只是力量碾压？ |
| D9 | 文化质感 | 文化元素的内化程度 | 文化是贴皮还是内化？是否增加了叙事厚度？ |
| D10 | 爽点工艺 | 读者满足感的精密设计 | 爽点是否有创意、有层次、不只是力量展示？ |

### D1: 信息流管理

**核心概念**

故事本质上是一台信息管理机器。
作者控制"读者知道什么"和"读者不知道什么"之间的差距，
用这个差距驱动好奇心、惊讶、重新理解等阅读体验。

**好的信息流 = 精密地控制读者的认知状态。**

**评分标准**

**1-2 分：信息线性释放**
  - 信息按时间顺序平铺，无悬念管理
  - 读者从头到尾知道的和角色一样多
  - 无伏笔，无信息缺口
  - 典型表现：旁白式叙事，"然后...然后...然后..."

**3-4 分：有基础悬念但管理粗糙**
  - 有简单的悬念（"他能打败敌人吗？"）
  - 有伏笔但回收率低（< 50%），或回收方式平淡
  - 信息释放节奏单一（一直慢或一直快）
  - 典型表现：套路化的"黄金三章"悬念，后续悬念管理松散

**5-6 分：有系统化的信息管理**
  - 有明确的伏笔-回收系统，回收率 > 70%
  - 信息释放有节奏变化（紧张→释放→再紧张）
  - 至少有 1-2 个"揭示改变了读者理解"的时刻
  - 典型表现：类型小说中信息管理较好的作品

**7-8 分：信息级联精密**
  - 每次揭示同时解答旧问题+提出新问题（信息级联）
  - 多条信息线交织，相互影响
  - 读者不断被"重新定位"——以为自己理解了，但揭示了新层面
  - 典型表现：进击的巨人（墙外真相→马莱→地鸣的级联）

**9-10 分：叙事重构级别**
  - 揭示改变了整个故事的理解框架
  - 读者需要从头重新理解之前的所有内容
  - 信息管理系统本身就是叙事的核心驱动力
  - 典型表现：进击的巨人马莱篇（发现自己一直在"敌方视角"）、
    道诡异仙（"现实"本身被质疑）

**边界案例判断规则**

  Q: 伏笔回收了但方式很平淡，算不算？
  A: 回收本身算 +1，但平淡的回收不加分。
     "回收方式"单独评估：是否出乎意料、是否改变了理解。

  Q: 信息级联和叙事重构怎么区分？
  A: 级联 = 新问题替代旧问题（认知被更新）
     重构 = 整个理解框架被推翻（认知被颠覆）
     如果只是"原来XX是YY"→ 级联
     如果是"原来整个故事不是我以为的那样"→ 重构

  Q: 悬念很多但都没解决，怎么评？
  A: 未解决的悬念是负面指标。
     张力曲线应该有解决周期，不是一直升高。
     如果 Q_pending 一直 > Q_resolved → 扣分（挖坑不填）

### D2: 角色弧光

**核心概念**

角色弧光 = 角色从故事开始到结束的变化轨迹。
好的弧光不是"变强了"，而是"在压力下做出选择，选择导致不可逆的变化"。

**关键判断：变化是"故事驱动"还是"作者强行"？**

**评分标准**

**1-2 分：扁平角色/工具人**
  - 角色从头到尾没有变化
  - 角色没有独立动机，只是服务于情节的功能
  - 去掉名字无法区分不同角色
  - 典型表现：大部分配角是"功能性角色"（提供信息、提供帮助、提供冲突）

**3-4 分：有变化但不自然**
  - 角色有变化，但变化缺乏铺垫（"突然顿悟"）
  - 变化是线性的（一直变好或一直变坏），没有内心挣扎
  - 配角完全没有弧光
  - 典型表现：主角"升级"式的成长（打怪→变强→打更强的怪）

**5-6 分：有合理弧光，但深度一般**
  - 变化有事件驱动，有因果链
  - 有基本的内心冲突（想做A但不得不做B）
  - 至少 1-2 个配角有独立弧光
  - 典型表现：类型小说中角色塑造较好的作品

**7-8 分：弧光深刻且与主题耦合**
  - 角色的核心冲突与故事主题直接相关
  - 变化是渐进的、有层次的（不是一次性转变）
  - 多个主要角色有独立弧光
  - 弧光类型多样（正向/负向/扁平/颠覆）
  - 典型表现：死亡笔记（夜神月的渐进堕落）、猎人（小杰的代价）

**9-10 分：弧光是故事的核心驱动力**
  - 角色的变化就是故事本身
  - 删掉角色弧光，故事就不成立
  - 弧光深刻到读者能在角色身上看到自己
  - 典型表现：钢炼（爱德华对"等价交换"的理解就是整个故事）、
    进击的巨人（艾伦的弧光就是"自由的代价"）

**边界案例判断规则**

  Q: 主角没变但配角变了，怎么评？
  A: 主角的"扁平弧光"（信念型）如果处理得好，可以是高分。
     例：JOJO 的空条承太郎——信念不变但理解深化。
     关键看"不变"是否也是一种有意义的选择。

  Q: 角色变化太快（一章就转变），算不算？
  A: 快速转变本身不是问题，关键是是否有足够的铺垫。
     如果铺垫充分（前文有大量暗示），一章转变可以是高分。
     如果没有铺垫，直接扣分。

  Q: 反派的弧光怎么评估？
  A: 反派的弧光是独立维度。
     好的反派有自己的动机和弧光（莱纳在进击的巨人中）。
     烂的反派是纯粹的"邪恶存在"，没有动机。

### D3: 世界观构建

**核心概念**

世界观不是"设定集"——它是一个"可能性空间"。
好的世界观：从少量基础规则推导出丰富的可能性。
烂的世界观：大量零散设定，互相不关联，随时可以打破。

**评分标准**

**1-2 分：设定稀薄/随意**
  - 世界是"默认奇幻世界"，没有独特设定
  - 规则随时可以被打破（"反正作者说了算"）
  - 设定通过大段旁白解释

**3-4 分：有基本设定但不成体系**
  - 有一些有趣的设定，但缺乏系统性
  - 设定之间没有关联（力量体系和政治体系无关）
  - 存在"Deus Ex Machina"（突然出现新能力解决危机）

**5-6 分：有自洽的系统**
  - 核心规则清晰，始终被遵守
  - 有基本的推导关系（规则→后果）
  - 设定通过行动展示，但深度一般

**7-8 分：系统精密且有深度**
  - 多个系统相互关联（力量体系↔政治体系↔经济体系）
  - 规则有"天花板"，不无限膨胀
  - 限制和代价产生叙事张力
  - 典型表现：凡人修仙传（修仙体系有清晰的等级和代价）

**9-10 分：世界观是叙事的骨架**
  - 删掉世界观，故事就不成立
  - 规则的限制本身就是故事冲突的来源
  - 世界观通过行动和后果展示，读者自然理解规则
  - 典型表现：钢炼（等价交换）、诡秘之主（22途径体系）

**边界案例判断规则**

  Q: 设定很多但很散，怎么评？
  A: 设定数量不是加分项，系统性才是。
     100个不相关的设定 < 10个相互关联的设定。

  Q: 力量体系有等级但不断加新等级，怎么评？
  A: 这是"力量膨胀"，负面指标。
     好的力量体系有明确的天花板。
     不断加新等级 = 作者没有规划 = 扣分。

### D4: 主题共振

**核心概念**

主题不是"故事告诉读者什么道理"。
主题是"故事的结构本身在说什么"。

**关键区分：**
  烂的主题表达：角色说教（"我明白了，原来XX是YY"）
  好的主题表达：结构即主题（删掉任何一部分，主题都不完整）

**评分标准**

**1-2 分：无主题/纯娱乐**
  - 故事只有事件，没有意义层面
  - 不存在可以讨论的深层含义

**3-4 分：有简单主题但处理粗糙**
  - 善恶对立、正义必胜
  - 主题通过角色嘴巴直接说出来
  - 典型表现："这个世界需要爱和和平"

**5-6 分：有主题且基本表达合理**
  - 有明确的控制思想
  - 主题通过情节展示，但偶尔有说教
  - 有基本的对立面（正方 vs 反方的观点都有合理性）

**7-8 分：主题深刻且表达自然**
  - 主题有多层含义
  - 主题通过结构展示（不说教）
  - 角色的核心冲突与主题直接相关
  - 读者需要思考才能理解主题的全部含义

**9-10 分：主题与结构完美统一**
  - 删掉任何一部分，主题都不完整
  - 主题从第一页就存在，贯穿到结尾
  - 主题是开放的，不同读者可以有不同的理解
  - 典型表现：钢炼（等价交换=力量体系=哲学主题=角色弧光）

**边界案例判断规则**

  Q: 故事很感人但没有明确主题，怎么评？
  A: "感人"不是主题。感人是效果，主题是意义。
     如果感人但没有可讨论的深层含义 → 主题维度分数低。
     但"感人"本身可能在 D10（爽点工艺）中加分。

  Q: 主题是"自由"，但故事里没有真正讨论"自由"的代价，怎么评？
  A: 主题的名字不是关键，关键是否被深入讨论。
     如果只是表面提到了"自由"但没有深入 → 5-6分。
     如果深入讨论了"自由"的多个层面 → 7-8分。

### D5: 结构张力

**核心概念**

故事的"好看"本质上是张力的管理——
什么时候紧张，什么时候释放，什么时候反转。

**评分标准**

**1-2 分：无张力/节奏平坦**
  - 故事是事件的线性排列
  - 没有冲突升级（后面的冲突不比前面更有挑战）
  - 没有真正的代价（主角总能化险为夷）

**3-4 分：有基本张力但单调**
  - 有冲突但节奏单一（一直在高潮或一直在平淡）
  - 冲突是重复的（同一类型的冲突反复出现）
  - 代价是假的（看似有损失但很快就恢复）

**5-6 分：有合理节奏**
  - 高潮和缓冲有交替
  - 冲突有基本的升级（赌注在增加）
  - 有至少一次"真正的代价"（不可逆的损失）

**7-8 分：多层次张力管理**
  - 章节级小波动 + 卷级中波动 + 全局大波动
  - 冲突在多个维度升级（赌注、复杂度、代价）
  - 节奏本身传达情感（紧张段落的节奏让人紧张）

**9-10 分：张力是叙事的核心引擎**
  - 张力曲线本身成为叙事的一部分
  - 读者能感受到作者对节奏的精确控制
  - 每个"释放"都是为了下一个更高的"紧张"
  - 典型表现：进击的巨人（每一季的张力都在递进）

### D6: 颠覆与反转

**核心概念**

反转不是"意想不到"——那太容易了（"突然天降陨石"也意想不到）。
反转 = 意料之外 + 情理之中 + 改变了理解。

**评分标准**

**1-2 分：无反转/完全可预测**
  - 读者从第一章就知道结局
  - 没有任何"意料之外"的时刻

**3-4 分：有反转但依赖信息隐瞒**
  - 反转是通过"隐藏信息"实现的（作者没告诉你关键信息）
  - 反转后回看，前面的伏笔不够充分
  - 典型表现："原来他一直在暗中观察"——但前文没有暗示

**5-6 分：反转有伏笔支撑**
  - 反转有伏笔，回看时能理解
  - 反转是"合理的意外"
  - 但反转没有改变对整个故事的理解

**7-8 分：反转改变了理解框架**
  - 反转迫使读者重新理解之前的内容
  - 反转不是单一事件，而是整个视角的转变
  - 典型表现：死亡笔记L的死亡（改变了整个故事的走向）

**9-10 分：反转是叙事的核心驱动力**
  - 反转本身就是故事的核心机制
  - 每次反转都深化了主题
  - 典型表现：进击的巨人（马莱篇的视角反转）
    道诡异仙（整个现实层面的质疑）

### D7: 展示艺术

**核心概念**

"Show, don't tell" 的本质：
通过行动、对话、细节让读者自己得出结论，
而不是通过旁白告诉读者应该得出什么结论。

**评分标准**

**1-2 分：纯旁白叙事**
  - 一切通过旁白解释（"他很愤怒"、"这个世界很残酷"）
  - 角色的情感通过旁白标注而非行动展示
  - 世界观通过"设定文档"式的旁白解释

**3-4 分：有展示但不一致**
  - 偶尔通过行动展示，但经常回退到旁白解释
  - 角色的内心独白重复了行动已经展示的内容
  - 典型表现："他紧握拳头（展示）。他非常愤怒（讲述）。"

**5-6 分：展示为主，讲述为辅**
  - 大部分通过行动和对话展示
  - 偶尔需要旁白解释复杂的世界观设定
  - 情感主要通过行动展示

**7-8 分：展示精湛**
  - 对话揭示性格，行动展示情感，细节传达氛围
  - 世界观通过角色的行动和后果自然展示
  - 读者通过积累细节自己得出结论

**9-10 分：展示即叙事**
  - 删掉所有旁白，读者仍然能理解故事
  - 每个细节都有叙事功能
  - 典型表现：JOJO 的视觉化叙事（用动作而非旁白传递信息）

### D8: 冲突设计

**核心概念**

冲突不是"打架"——冲突是"两个不可调和的力量碰撞"。
好的冲突：有策略性、有多层次、有真正的代价。
烂的冲突：纯力量碾压，无策略，无代价。

**冲突的四个层次：**
  内在冲突：角色与自己的信念/欲望的斗争
  人际冲突：角色之间的利益/理念冲突
  社会冲突：角色与体制/文化的冲突
  宇宙冲突：角色与命运/自然/超自然的冲突

**评分标准**

**1-2 分：无实质冲突/纯力量碾压**
  - 冲突只是"打一架"
  - 主角靠力量/运气赢
  - 没有策略，没有代价

**3-4 分：有基本冲突但缺乏层次**
  - 冲突主要在"人际"层面
  - 反派动机简单（"就是坏"）
  - 冲突的解决是"主角更强"

**5-6 分：冲突有层次和策略**
  - 至少有两个层次的冲突同时存在
  - 冲突的解决需要策略，不只是力量
  - 有基本的代价（赢了但失去了什么）

**7-8 分：冲突多元且深刻**
  - 四个层次的冲突都有涉及
  - 冲突的解决揭示了新信息或深化了主题
  - 每个冲突都有真正的赌注
  - 典型表现：猎人（蚁王篇——内在冲突+人际冲突+宇宙冲突）

**9-10 分：冲突是叙事的核心引擎**
  - 冲突本身推动角色成长和主题深化
  - 冲突的解决方式是"意料之外，情理之中"
  - 典型表现：JOJO（替身战斗=策略博弈+角色意志力展示）

---

### D9: 文化质感

**核心概念**

文化元素不是"贴皮"（换个名字就变成异世界）。
文化质感 = 文化元素内化为叙事逻辑的一部分。

**评分标准**

**1-2 分：无文化质感/纯架空**
  - 世界是"默认奇幻"，没有任何独特的文化元素
  - 或：有文化元素但是表面的（名字像但行为不像）

**3-4 分：文化作为背景**
  - 有文化设定但不影响情节
  - 文化元素是装饰性的（节日、服饰、建筑）

**5-6 分：文化影响情节**
  - 文化设定影响角色的行为和选择
  - 世界观的文化层面是自洽的

**7-8 分：文化内化为叙事逻辑**
  - 文化元素驱动冲突（不是装饰）
  - 世界观的文化层面与力量体系/政治体系交织
  - 文化元素有内在逻辑（不是随意堆砌）

**9-10 分：文化是叙事的骨架**
  - 删掉文化元素，故事就不成立
  - 文化元素本身产生叙事张力
  - 典型表现：JOJO（音乐/艺术文化内化为命名和美学系统）

---

### D10: 爽点工艺

**核心概念**

爽点 = 读者预期被满足或被超越的瞬间。
好的爽点：过程有创意，满足有层次。
烂的爽点：纯力量碾压，无过程。

**评分标准**

**1-2 分：无爽点/纯力量展示**
  - 主角赢了但过程无聊（"一拳秒杀"）
  - 爽点是重复的（每次都是同样的套路）

**3-4 分：有爽点但套路化**
  - 爽点是"废柴逆袭"的模板
  - 过程是可预测的（读者知道主角会赢）
  - 满足感单一（只是"赢了"）

**5-6 分：爽点有基本创意**
  - 赢的方式有策略性
  - 过程有悬念（不确定怎么赢）
  - 满足感有层次（赢了+揭示了新信息）

**7-8 分：爽点精湛**
  - 赢的方式出乎意料但合理
  - 满足感多层次（赢+揭示+情感冲击）
  - 爽点本身推动了情节
  - 典型表现：JOJO（战斗的创意性）、诡秘之主（信息差碾压）

**9-10 分：爽点是叙事的一部分**
  - 删掉爽点，故事就不完整
  - 爽点深化了主题
  - 典型表现：钢炼（爱德华的每一次炼金术使用都深化"等价交换"主题）

---

## 四、Judge 行为规范

### 4.1 评分流程

对于每个评估维度，Judge Agent 必须：

```
Step 1: 阅读提供的文本样本
Step 2: 在文本中寻找该维度的证据
Step 3: 对照评分标准，确定分数区间
Step 4: 在分数区间内，根据边界案例规则确定精确分数
Step 5: 输出结构化评分结果
```

### 4.2 输出格式

```json
{
  "dimension": "D1_信息流管理",
  "score": 7,
  "score_range": "7-8分: 信息级联精密",
  "evidence": [
    {
      "location": "第3章，第15段",
      "text_excerpt": "（引用原文50-100字）",
      "analysis": "这里埋下了一个伏笔：XX。这个伏笔在第45章被回收。"
    },
    {
      "location": "第45章，第8段",
      "text_excerpt": "（引用原文50-100字）",
      "analysis": "伏笔回收，且回收方式出乎意料——原来XX是YY。"
    }
  ],
  "reasoning": "作品有系统化的伏笔管理，回收率高，且至少有一次信息级联（揭示同时解答旧问题+提出新问题）。但未达到9-10分的'叙事重构'级别，因为没有一次揭示彻底改变了对整个故事的理解。",
  "borderline_notes": "考虑给6分，因为信息级联的深度还差一点。但回收率和伏笔密度支持7分。"
}
```

### 4.3 关键规则

  1. 必须引用具体文本证据，不能凭感觉打分
  2. 每个评分必须说明为什么是这个分数而不是相邻的分数
  3. 如果两个分数都可能，选择更接近边界案例规则的那个
  4. 不允许因为一个维度好就默认另一个维度也好
  5. 评分必须基于提供的文本，不能假设未提供的内容

## 五、文本采样策略

### 5.1 采样原则

网文百万字，不可能全读。但不同维度需要不同类型的样本。

### 5.2 采样方案

```
采样类型 A：关键节点采样
  用途：角色弧光、主题共振、反转评估
  内容：开头5章 + 每个重大转折点前后各2章 + 结尾5章
  预计：20-30个片段，每个3000-5000字

采样类型 B：连续章节采样
  用途：节奏分析、展示艺术、冲突设计
  内容：选取3-5个连续章节（每个1-2万字）
  位置：前期、中期、后期各一段
  预计：3-5个片段，每个10000-15000字

采样类型 C：分散采样
  用途：世界观构建、文化质感、信息流管理
  内容：每10%的文本取一个3000字片段
  预计：10个片段，每个3000字

采样类型 D：全书索引
  用途：伏笔回收率、连载稳定性
  内容：每章取首段和末段
  预计：全书章节数 × 2段
```

## 六、验证策略

### 6.1 标杆作品集

选择 20 部作品作为验证集：
  S级（7部）：诡秘之主、钢炼、进击的巨人、JOJO、道诡异仙、猎人、死亡笔记
  A级（7部）：斗破苍穹（前中期）、完美世界、一拳超人、海贼王、绍宋、惊悚乐园、命运石之门
  B级（6部）：典型系统文、典型退婚流、典型签到文、典型后宫文、典型模板化玄幻、典型烂尾文

### 6.2 人类共识

每部作品收集 3-5 位资深读者的评分作为人类共识。
如果 Agent 评分与人类共识的 Spearman ρ > 0.7，认为 Agent 可信。

### 6.3 迭代流程

```
Round 1: 用当前 Rubric 评估标杆作品
    ↓
Round 2: 对比 Agent 评分 vs 人类共识
    ↓
Round 3: 分析差异最大的案例
    ↓
Round 4: 修正 Rubric（调整评分标准、增加边界案例规则）
    ↓
Round 5: 重新评估，检查改进
    ↓
重复直到 ρ > 0.7
```
